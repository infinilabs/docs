# Getting Started

INFINI Loadgen is a lightweight HTTP load generator and testing suite designed for Easysearch, Elasticsearch, and OpenSearch.

**Features:**

- Single binary, no dependencies, cross-platform
- Intuitive DSL syntax with template-based parameter randomization
- High concurrency and fixed QPS control
- Built-in assertions and dynamic variable registration for multi-step test orchestration
- Suitable for performance benchmarking, functional validation, and CI integration
