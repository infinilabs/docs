# 入门指南

INFINI Loadgen 是一款专为 Easysearch、Elasticsearch、OpenSearch 设计的轻量级 HTTP 性能测试与集成测试工具。

**特性：**

- 单文件、无依赖、跨平台
- DSL 语法直观，支持模板化参数随机
- 支持高并发与固定 QPS 控制
- 内置断言与动态变量注册，可编排多步骤测试
- 可用于性能压测、功能验证与 CI 集成
