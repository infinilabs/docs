      INFINI Agent #

      介绍 #

      INFINI Agent 是一个强大的开源管理代理，专为 Easysearch、Elasticsearch 和 OpenSearch 多实例环境设计。它高效地收集和上传搜索集群的日志和指标，提供全面的监控和可观测性。该代理可通过 INFINI Console 或 INFINI Cloud 进行管理，支持所有主流操作系统和平台。凭借其轻量级架构和零外部依赖，INFINI Agent 能够实现快速、无忧的部署。

      特性 #

      • 轻量高效：内存占用低于 10MB，运行时 CPU 使用率不到 1%
      • 全面监控：收集搜索集群的健康信息、集群统计、索引统计和节点统计
      • 日志管理：捕获并上传包括 Elasticsearch 在内的集群节点日志
      • 实时日志：提供对目标主机日志的实时访问，无需登录服务器或手动检索日志文件
      • 系统指标：监控并报告主机级别的性能指标
      • 多平台支持：适用于所有主流操作系统
      • 零依赖：自包含安装，无需外部依赖
