      es_cluster_health #

      描述 #

      采集集群健康指标。

      配置示例 #

      pipeline
  - name: collect_default_es_cluster_health
    auto_start: true
    keep_running: true
    retry_delay_in_ms: 3000
    processor:
    - es_cluster_health:
        elasticsearch: default

      参数说明 #

      名称类型说明
      elasticsearchstring集群实例名称（请参考 elasticsearch 的 name 参数）
      labelsmap自定义标签
