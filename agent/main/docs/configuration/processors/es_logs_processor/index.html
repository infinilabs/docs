    es_logs_processor #

    Description #

    Collect the cluster log files.

    Configuration Example #

    pipeline
  - name: collect_default_es_logs
    auto_start: true
    keep_running: true
    retry_delay_in_ms: 3000
    processor:
    - es_logs_processor:
        elasticsearch: default
        logs_path: "/opt/dev-environment/ecloud/logs"
        queue_name: logs

    Parameter Description #

    NameTypeDescription
    queue_namestringLog collection queue name
    elasticsearchstringCluster instance name (Please see elasticsearch name parameter)
    logs_pathstringCluster log path
    labelsmapCustom labels
