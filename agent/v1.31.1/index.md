# INFINI Agent

## Introduction

**INFINI Agent** is a powerful open-source management agent designed for Easysearch, Elasticsearch, and OpenSearch multi-instance environments. It efficiently collects and uploads logs and metrics from your search clusters, providing comprehensive monitoring and observability. The agent can be managed through INFINI Console or INFINI Cloud and supports all mainstream operating systems and platforms. With its lightweight architecture and zero external dependencies, INFINI Agent enables quick and hassle-free deployment.

## Features

- **Lightweight & Efficient**: Memory footprint under 10MB with less than 1% CPU usage during operation
- **Comprehensive Monitoring**: Collects health information, cluster stats, index stats, and node stats from search clusters
- **Log Management**: Captures and uploads logs from cluster nodes including Elasticsearch and compatible engines
- **Real-time Logging**: Provides real-time access to logs on target hosts, eliminating the need to log into servers or manually retrieve log files
- **System Metrics**: Monitors and reports host-level performance metrics
- **Multi-Platform Support**: Works across all mainstream operating systems
- **Zero Dependencies**: Self-contained installation with no external requirements
