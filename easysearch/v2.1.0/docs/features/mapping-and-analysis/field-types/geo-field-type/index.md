# Geographic 地理字段类型

Geographic 地理字段包含表示地图上的点或形状的值。下表列出了 Easysearch 支持的所有地理字段类型。

| 字段数据类型 | 描述                               |
| ------------ | ---------------------------------- |
| geo_point    | 由纬度和经度指定的地理点。         |
| geo_shape    | 地理形状，如多边形或地理点的集合。 |

两种字段类型都支持 [Geohash 编码]({{< relref "geohash" >}})，用于高效地索引和聚合地理位置数据。
