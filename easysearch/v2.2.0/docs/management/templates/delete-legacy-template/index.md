# 删除索引模板（旧版）

当旧版索引模板不再需要时，可以通过 UI 进行删除操作。

## 操作步骤

1. **进入模板页面**：在左侧导航栏点击「模板」，切换至「索引模板（旧版）」标签页。
2. **打开操作菜单**：找到需要删除的模板，点击「操作」列的更多按钮，展开操作菜单。
3. **选择删除操作**：在菜单中点击「删除」选项，弹出删除确认提示框。

{{% load-img "/img/management/templates/delete-legacy-template/image-1.png" %}}

4. **确认删除**：在提示框中点击「确定」按钮，完成模板删除。

{{% load-img "/img/management/templates/delete-legacy-template/image-2.png" %}}

5. **验证删除结果**：删除成功后，该模板将从列表中消失，代表删除操作完成。

{{% load-img "/img/management/templates/delete-legacy-template/image-3.png" %}}

## 注意事项

- 删除模板**不会影响**已通过该模板创建的索引，已有索引的设置和映射保持不变。
- 删除操作不可撤销，如需恢复请重新创建模板。
