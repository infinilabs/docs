# 更改表格字段列位置

可以通过拖拽或其他方式调整表格中字段列的显示顺序。

## 操作步骤

1. **定位表格字段**：在「数据探索」页面的数据表格中，找到需要移动的目标字段列。
2. **拖动字段列**：将鼠标悬停在目标字段的表头上，按住鼠标左键并拖动到目标位置后松开，即可完成字段列位置的调整。

{{% load-img "/img/management/data-explorer/change-column-position/image-1.png" %}}

3. **查看调整结果**：拖动完成后，表格列顺序将立即更新。
