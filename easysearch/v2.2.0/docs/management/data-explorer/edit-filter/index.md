# 编辑过滤条件

可以对已添加的过滤条件进行修改，以调整数据筛选规则。

## 操作步骤

1. **唤起操作菜单**：在数据探索页面，找到已生效的过滤条件标签（如 `_id:LJAUJ0BlYzaDM3kYDbw`），点击该标签，弹出操作菜单。
2. **进入编辑页面**：在菜单中选择「编辑过滤条件」，打开过滤条件配置窗口。

{{% load-img "/img/management/data-explorer/edit-filter/image-1.png" %}}

3. **修改过滤参数**：
   - 可按需调整「字段」「运算」的选择项；
   - 在「值」输入框中，修改目标过滤值（示例中修改为 `nYsUlJ0Bc_MPNLBvYYQh`）。

{{% load-img "/img/management/data-explorer/edit-filter/image-2.png" %}}

4. **（可选）高级编辑**：如需自定义复杂查询，可点击窗口右上角的「编辑 Query DSL」，直接编写 Elasticsearch 查询语句。
5. **保存并生效**：配置完成后，点击「保存」按钮，系统将自动更新过滤条件。
6. **查看结果**：更新生效后，搜索栏的过滤标签会同步更新，右侧数据列表将展示符合新条件的文档记录。

{{% load-img "/img/management/data-explorer/edit-filter/image-3.png" %}}
