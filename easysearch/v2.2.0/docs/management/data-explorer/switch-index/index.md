# 切换索引

在数据探索页面中，可以切换当前查询的目标索引。

## 操作步骤

1. **打开索引选择下拉框**：在数据探索页面顶部，点击当前索引名称（示例中为 `nginx_zstd-1`）旁的下拉箭头，展开索引选择列表。
2. **选择目标索引**：在展开的列表中，点击需要切换的目标索引（如 `.security`、`.analysis_ik` 等），列表会同步展示各索引的存储占用、文档数等信息。
3. **完成切换**：选中目标索引后，页面将自动加载该索引下的所有文档数据，完成索引切换。
4. **快速检索索引**：可在列表顶部的搜索框中输入索引名称，快速筛选定位目标索引，提升切换效率。

## 注意事项

- 切换索引后，当前已设置的过滤条件和搜索语句将被清空，如有需要请提前记录查询条件。
- 索引需包含时间字段才能使用时间范围筛选功能，无时间字段的索引将无法按时间过滤数据。
