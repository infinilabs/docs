# 暂停创建进群

在集群创建过程中，可以暂停当前的创建流程。

## 操作步骤

1. **进入服务管理页面**：在 Agent UI 的「服务管理」页面中，找到状态为「创建中」的目标集群。

2. **打开暂停操作菜单**：点击目标集群操作列的「...」按钮，在下拉菜单中选择「暂停」选项。

{{% load-img "/img/management/service-management/pause-cluster-creation/image-1.png" %}}

3. **确认暂停操作**：在弹出的确认窗口中，点击「确定」按钮，暂停该创建任务。

{{% load-img "/img/management/service-management/pause-cluster-creation/image-2.png" %}}

4. **验证暂停结果**：操作完成后，服务列表中该集群的状态将更新为「创建暂停」，代表暂停成功。

{{% load-img "/img/management/service-management/pause-cluster-creation/image-3.png" %}}

## 注意事项

- 暂停操作仅对「创建中」状态的集群生效，已创建完成的集群无法执行此操作。
- 暂停后可随时通过操作菜单恢复创建任务，暂停期间不会影响其他已运行的集群服务。
- 暂停状态下的创建任务不会自动继续，需手动恢复操作才能完成集群部署。
