本部分收录 Easysearch 的版本发布记录，包括每个版本的重要变更、新增功能与不兼容调整。

适合在以下场景查阅：

- 准备升级集群时，了解从当前版本到目标版本之间的改动
- 排查行为差异时，确认某个特性是在什么时候引入或修改的

建议搭配使用：

- 当你只想“了解当前能做什么”时，请优先阅读各功能章节与参考手册
- 当你要**从旧版本迁移/升级**，或对具体版本行为有疑问时，再回到本章节按版本对照查看
