本页列出自动补全字段类型的索引。  
如何选择自动补全方案、设计搜索体验，请优先阅读搜索章节的 Guide。

- [部分匹配]({{< relref "/docs/features/fulltext-search/partial-matching.md" >}})

## 字段类型总览

下表列出了 Easysearch 支持的所有自动补全字段类型。
| 字段数据类型 | 描述 |  
|----------------------------|---------------------------------------------------------------------------------------------------------|  
| **completion** | 使用前缀补全提供自动补全功能。在使用此功能之前，需要将所有可能的补全项上传到索引中。 |  
| **search_as_you_type** | 提供 search_as_you_type 的功能，支持前缀补全和中缀补全。 |
