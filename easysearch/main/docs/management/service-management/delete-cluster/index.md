# 删除集群

可以删除不再需要的集群，释放相关资源。

## 操作步骤

1. **进入服务管理页面**：在 Agent UI 中进入「服务管理」页面，查看已安装的 Easysearch 服务列表。

2. **打开删除操作菜单**：在目标服务的操作列，点击「...」按钮，在下拉菜单中选择「删除」选项。

{{% load-img "/img/management/service-management/delete-cluster/image-1.png" %}}

3. **确认删除操作**：在弹出的删除确认窗口中，确认要删除的节点名称，点击「确定」按钮。

{{% load-img "/img/management/service-management/delete-cluster/image-2.png" %}}

4. **验证删除结果**：删除完成后，服务列表中将不再显示该节点，代表删除操作成功。

{{% load-img "/img/management/service-management/delete-cluster/image-3.png" %}}

## 注意事项

- 删除操作会**永久移除该服务并同步删除对应数据目录**，数据将无法恢复，请谨慎操作。
- 建议在执行删除前，先停止服务并备份必要数据。
- 如果该节点是集群中的唯一节点，删除后整个集群也将被移除。
