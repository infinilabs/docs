# 进入 Easysearch UI 节点详情

可以从服务管理平台快速跳转到 Easysearch UI 的节点详情页面。

## 操作步骤

1. **进入服务管理页面**：在 Agent UI 的「服务管理」页面中，找到目标节点，点击节点名称。

{{% load-img "/img/management/service-management/enter-ui-node-details/image-1.png" %}}

2. **完成身份验证**：在跳转的登录页面中，输入集群地址、用户名和密码，点击箭头按钮进行登录。

{{% load-img "/img/management/service-management/enter-ui-node-details/image-2.png" %}}

3. **进入节点详情页面**：登录成功后，将直接进入该节点的详情页面，查看节点概览、分片、配置、插件及日志等信息。

{{% load-img "/img/management/service-management/enter-ui-node-details/image-3.png" %}}
