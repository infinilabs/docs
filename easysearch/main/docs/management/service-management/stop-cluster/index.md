# 停止集群

可以停止当前正在运行的集群。

## 操作步骤

1. **进入服务管理页面**：登录 Agent UI 后，在服务管理页面的「服务列表」中，找到目标集群。

2. **触发停止操作**：点击目标集群「操作」列的更多按钮，在下拉菜单中选择「停止」选项。

{{% load-img "/img/management/service-management/stop-cluster/image-1.png" %}}

3. **确认停止操作**：在弹出的「停止」确认窗口中，核对节点名称，点击「确定」按钮。

{{% load-img "/img/management/service-management/stop-cluster/image-2.png" %}}

4. **验证停止结果**：系统执行停止操作后，返回服务列表页面，目标集群的状态将更新为「已停止」，表示集群已成功关闭。

{{% load-img "/img/management/service-management/stop-cluster/image-3.png" %}}

## 注意事项

- 停止集群会中断所有正在运行的业务请求，操作前请确认业务已暂停或已切换至其他可用集群。
- 停止操作不会删除集群配置和数据，仅停止进程，后续可通过「启动」操作恢复运行。
- 停止过程中请勿关闭页面或中断操作，避免出现进程状态异常的情况。
