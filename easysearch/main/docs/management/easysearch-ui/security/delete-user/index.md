# 删除用户

## 操作步骤

1. **进入安全管理页面**：在左侧导航栏点击「安全管理」，默认进入「用户」标签页，查看现有用户列表。

2. **打开操作菜单**：找到需要删除的目标用户（如 `test_user`），点击该行「操作」列的更多按钮（`...`），在弹出菜单中选择「删除」。

![](/img/easysearch-ui/security/delete-user/image-1.png)

3. **确认删除操作**：在弹出的确认提示框中，点击「确定」，系统将执行用户删除。

![](/img/easysearch-ui/security/delete-user/image-2.png)

4. **查看操作结果**：操作完成后，该用户将从列表中移除，代表删除成功。

![](/img/easysearch-ui/security/delete-user/image-3.png)

## 注意事项

- **请勿删除管理员账户**（如 `admin`），否则可能导致无法登录管理界面或无法执行管理操作。
- 删除用户后，该用户的所有活跃会话将立即失效。
- 用户删除操作不可撤销，如需恢复请重新创建用户并分配相同角色。
