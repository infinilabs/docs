# 删除角色

## 操作步骤

1. **进入角色管理页**：在左侧导航栏点击「安全管理」，切换到「角色」标签页，查看现有角色列表。

2. **打开操作菜单**：找到需要删除的目标角色（如 `test_role`），点击该行「操作」列的更多按钮（`...`），在弹出菜单中选择「删除」。

![](/img/easysearch-ui/security/delete-role/image-1.png)

3. **确认删除操作**：在弹出的确认提示框中，点击「确定」，系统将执行角色删除。

![](/img/easysearch-ui/security/delete-role/image-2.png)

4. **查看操作结果**：操作成功后，该角色将从角色列表中移除，列表中不再显示该角色，代表删除完成。

![](/img/easysearch-ui/security/delete-role/image-3.png)

## 注意事项

- 删除角色前，请确认该角色**未被任何用户引用**，否则相关用户将失去该角色所赋予的权限，可能导致业务访问异常。
- 内置系统角色不可删除，仅自定义角色支持删除操作。
