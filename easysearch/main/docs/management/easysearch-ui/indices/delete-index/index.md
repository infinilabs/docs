# 删除索引

删除操作会永久移除索引及其所有数据，**不可恢复**，请谨慎操作。

## 操作步骤

1. **进入索引列表**：在「索引」页面，找到需要删除的目标索引（如 `testing_index_creation-clone`）。

2. **打开操作菜单**：点击该索引行「操作」列的更多按钮（`...`），在下拉菜单中选择「删除索引」。

![](/img/easysearch-ui/indices/delete-index/image-1.png)

3. **确认删除**：在弹出的确认提示框中，点击「确定」，系统将执行索引删除操作。

![](/img/easysearch-ui/indices/delete-index/image-2.png)

4. **查看结果**：操作完成后，索引列表中将不再显示该被删除的索引（如 `testing_index_creation-clone` 已从列表中移除），即代表删除成功。

![](/img/easysearch-ui/indices/delete-index/image-3.png)

## 注意事项

> **⚠️ 警告**：删除索引是**不可逆操作**，索引的所有数据、Mapping 和设置将被永久移除，且无法通过任何方式恢复。执行前请务必确认：
>
> - 该索引的数据已不再需要，或已通过快照（Snapshot）等方式完成备份。
> - 没有应用或别名正在引用该索引。

- 删除系统索引（如以 `.` 开头的索引）可能导致集群功能异常，请谨慎操作。
- 如果只是暂时不需要该索引，建议使用「关闭索引」替代删除操作。
