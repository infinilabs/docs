# 修改管道

当摄取管道的名称、描述或处理器配置需要调整时，可以通过管道编辑页面进行修改。

## 操作步骤

1. **进入管道管理页面**：在左侧导航栏点击「管道管理」，进入摄取管道列表页面。

2. **打开编辑页面**：找到需要修改的目标管道，点击「操作」列的更多按钮（`...`），在弹出菜单中选择「编辑」。

{{% load-img "/img/management/pipeline-management/edit-pipeline/image-1.png" %}}

3. **修改管道配置**：在「编辑摄取管道」页面中，按需修改管道名称、描述和处理器配置。示例中将名称修改为 `test_pipeline_2_modify`，并新增了 `Bytes` 处理器。

4. **保存修改**：确认配置无误后，点击「保存」按钮。

{{% load-img "/img/management/pipeline-management/edit-pipeline/image-2.png" %}}

5. **查看修改结果**：保存成功后，将返回管道管理列表。示例中管道名称已更新为 `test_pipeline_2_modify`，处理器数量变为 `2`。

{{% load-img "/img/management/pipeline-management/edit-pipeline/image-3.png" %}}

## 注意事项

- 修改处理器配置会影响后续通过该管道写入的数据处理结果，请在保存前确认处理器顺序和字段配置。
- 如管道已被索引写入流程引用，修改前建议确认业务侧是否依赖当前处理逻辑。
- 管道名称不可与已有管道重复，否则保存将失败。
