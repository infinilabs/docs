# 删除数据流

当数据流不再使用时，可以通过数据流列表进行删除。

## 操作步骤

1. **进入数据流页面**：在左侧导航栏点击「数据流」，进入数据流列表页面。

2. **选择删除操作**：找到需要删除的目标数据流，点击「操作」列的更多按钮（`...`），在弹出菜单中选择「删除」。

{{% load-img "/img/management/data-stream/delete-data-stream/image-1.png" %}}

3. **确认删除**：在弹出的确认提示框中，点击「确定」按钮，系统将执行删除操作。

{{% load-img "/img/management/data-stream/delete-data-stream/image-2.png" %}}

4. **查看删除结果**：删除成功后，该数据流将从列表中移除。示例中 `test_data_stream` 已不再显示在数据流列表中。

{{% load-img "/img/management/data-stream/delete-data-stream/image-3.png" %}}

## 注意事项

- 删除数据流会移除数据流配置及其关联关系，请在操作前确认业务不再通过该数据流写入数据。
- 如果业务仍使用该数据流名称写入数据，删除后相关写入请求可能失败。
- 删除操作不可撤销，如需恢复请重新创建数据流并确认关联模板配置。
