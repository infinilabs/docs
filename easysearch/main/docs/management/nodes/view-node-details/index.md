# 查看节点详情

通过节点详情页，可一站式查看节点的完整运行数据。

## 操作步骤

1. **进入节点列表页**：在左侧导航栏点击「节点」，进入节点管理列表页面，可查看集群内所有节点的基础运行信息。

2. **进入节点详情页**：点击目标节点的名称（如带星号标识的 `easysearch-9jtew92c978t-data0-0`），即可进入该节点的详情页面。

{{% load-img "/img/management/nodes/view-node-details/image-1.png" %}}

3. **查看核心信息**：在详情页中，可一站式查看该节点的完整运行数据，包括：
   - 基础信息：节点 ID、传输地址、节点角色、数据路径
   - 运行状态：运行时间、CPU 使用率、JVM 堆内存使用率
   - 存储信息：可用存储、已用存储、分片数、文档数
   - 性能指标：CPU 使用率趋势、JVM 堆内存用量、索引延迟、索引吞吐等
