# 表格字段值排序

可以按照某个字段的值对表格数据进行升序或降序排列。

## 操作步骤

1. **点击表头排序**：在「数据探索」页面的数据表格中，点击目标字段的表头名称，即可切换排序方式（升序↑ / 降序↓）。表头旁会出现箭头图标指示当前排序方向。

{{% load-img "/img/management/data-explorer/sort-table-field/image-1.png" %}}

## 注意事项

- 部分字段类型（如 `text`）默认不支持排序，需启用 `fielddata` 或使用对应的 `keyword` 子字段。
