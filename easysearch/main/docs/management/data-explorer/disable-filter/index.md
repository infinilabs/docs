# 暂时停用过滤条件

暂时停用过滤条件可以在不删除条件的情况下临时取消其过滤效果。

## 操作步骤

1. **唤起操作菜单**：在数据探索页面，找到已生效的过滤条件标签（如 `_id:nYsUlJ0Bc_MPNLBvYYQh`），点击该标签，弹出操作菜单。
2. **执行停用操作**：在菜单中选择「暂时停用」，系统将临时关闭该过滤条件的生效逻辑。

{{% load-img "/img/management/data-explorer/disable-filter/image-1.png" %}}

3. **查看生效结果**：
   - 过滤标签将变为灰色（示例中标签样式弱化显示），标识当前为停用状态；
   - 右侧数据列表将恢复展示该索引下的全部文档记录，过滤条件仅保留配置，未实际生效。

{{% load-img "/img/management/data-explorer/disable-filter/image-2.png" %}}

4. **恢复过滤（可选）**：如需重新启用，可再次点击该停用标签，选择「重新启用」即可恢复过滤逻辑。
