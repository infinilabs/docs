# 自动刷新数据

开启自动刷新后，数据探索页面将按设定的时间间隔自动更新查询结果。

## 操作步骤

1. **开启自动刷新**：在数据探索页面右上角，点击**播放图标**（▶️），即可启动自动刷新，图标同步变为暂停样式（⏸️）。

{{% load-img "/img/management/data-explorer/auto-refresh/image-1.png" %}}

{{% load-img "/img/management/data-explorer/auto-refresh/image-2.png" %}}

2. **配置刷新间隔**：
   - 点击暂停图标旁的下拉箭头，打开时间设置菜单。
   - 在「刷新间隔」输入框中，填写自定义刷新时长（如示例中的10秒），选择时间单位（秒）。
   - 点击「应用」按钮，保存配置。

{{% load-img "/img/management/data-explorer/auto-refresh/image-3.png" %}}

3. **暂停自动刷新**：点击页面右上角的**暂停图标**（⏸️），即可停止自动刷新，图标恢复为播放样式。

## 注意事项

- 刷新间隔设置过短会增加集群查询负载，建议根据业务实际需要合理配置。
- 页面切换到后台时，自动刷新仍会持续执行，如不需要请及时暂停。
