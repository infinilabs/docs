# 快速新增字段值过滤条件

可以直接点击表格中的字段值，快速将其添加为过滤条件。

## 操作步骤

1. **展开行详情**：在「数据探索」页面的数据表格中，点击目标行左侧的「展开」图标（▶），展开完整的字段列表。

{{% load-img "/img/management/data-explorer/quick-add-field-filter/image-1.png" %}}

2. **选择过滤操作**：在展开的字段列表中，找到目标字段值，点击右侧的操作图标：
   - **「+」图标**：新增包含该字段值的过滤条件（is 类型）。
   - **「–」图标**：新增排除该字段值的过滤条件（is not 类型）。

{{% load-img "/img/management/data-explorer/quick-add-field-filter/image-2.png" %}}

3. **查看过滤结果**：新增的过滤条件将自动应用，搜索结果将被筛选，过滤条件显示在搜索栏下方。

{{% load-img "/img/management/data-explorer/quick-add-field-filter/image-3.png" %}}
