# 批量操作过滤条件

支持对多个过滤条件进行批量启用、停用或删除等操作。

## 操作步骤

1. **打开批量操作菜单**：在数据探索页面，点击过滤条件栏左侧的**批量操作图标**（三条横线带下拉箭头样式），弹出「变更所有过滤条件」菜单。

{{% load-img "/img/management/data-explorer/batch-filter/image-1.png" %}}

2. **选择批量操作**：点击目标操作，以「反转包含」为例：
   - 选择「反转包含」，系统将批量反转所有过滤条件的匹配逻辑。

{{% load-img "/img/management/data-explorer/batch-filter/image-2.png" %}}

3. **查看结果**：
   - 过滤标签新增 `'NOT'` 前缀，所有条件转为排除逻辑；
   - 右侧列表展示不满足原条件的文档，完成批量操作。

{{% load-img "/img/management/data-explorer/batch-filter/image-3.png" %}}

## 其他批量操作功能

- **全部启用**：批量恢复所有停用过滤条件
- **全部禁用**：批量暂停所有过滤条件
- **反转启用状态**：批量切换所有过滤条件的启用 / 停用状态
- **移除全部**：批量删除所有过滤条件

## 注意事项

- 「移除全部」操作不可撤销，执行后所有过滤条件将被永久删除，请谨慎使用。
- 批量操作会同时作用于所有已添加的过滤条件，包括当前已停用的条件。
