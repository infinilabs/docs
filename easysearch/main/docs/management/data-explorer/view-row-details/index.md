# 查看表格行完整数据

可以展开表格中的某一行，查看该文档的所有字段和完整数据。

## 操作步骤

1. **展开行详情**：在「数据探索」页面的数据表格中，点击目标行左侧的「展开」图标（▶），展开该行的完整数据详情。

{{% load-img "/img/management/data-explorer/view-row-details/image-1.png" %}}

2. **查看完整数据**：展开后，可以看到该行记录的所有字段和值，包括表格中未显示的字段。

{{% load-img "/img/management/data-explorer/view-row-details/image-2.png" %}}

3. **切换视图**：支持在「Table」（表格）和「JSON」视图之间切换，方便查看原始 JSON 格式的文档数据。

{{% load-img "/img/management/data-explorer/view-row-details/image-3.png" %}}
