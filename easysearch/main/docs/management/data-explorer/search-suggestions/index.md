# 搜索框智能提示

搜索框支持智能提示功能，可以在输入查询时自动补全字段名和值。

## 操作步骤

1. **激活搜索框**：在数据探索页面顶部的搜索输入框中点击，激活输入状态。
2. **获取字段智能提示**：激活后系统自动弹出下拉列表，展示当前索引下的所有可用字段（如 `_id`、`@timestamp`、`clientip` 等），点击目标字段可快速补全到搜索框。

{{% load-img "/img/management/data-explorer/search-suggestions/image-1.png" %}}

3. **获取语法智能提示**：在输入字段名后（如输入 `_id`），系统会自动弹出语法提示列表，包含查询运算符（`:`、`*`）、逻辑连接符（`and`、`or`）等，点击可快速补全查询语法。

{{% load-img "/img/management/data-explorer/search-suggestions/image-2.png" %}}

4. **执行查询**：完成查询语句编写后，点击「更新」按钮，系统将执行搜索并展示符合条件的文档记录。

## 注意事项

- 搜索框默认使用 KQL（Kibana Query Language）语法，也支持切换为 Lucene 查询语法。
- 使用通配符查询（如 `*`）时可能影响查询性能，建议尽量使用精确匹配或限定字段范围。
