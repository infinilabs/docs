# 导出 CSV 数据

可以将当前数据探索的查询结果导出为 CSV 文件，方便离线分析。

## 操作步骤

1. **查询目标数据**：在「数据探索」页面中，通过设置索引、时间范围、搜索条件和过滤器等，定位到需要导出的数据。
2. **点击导出按钮**：在表格工具栏中，点击「导出」按钮。

{{% load-img "/img/management/data-explorer/export-csv/image-1.png" %}}

3. **配置导出选项**：在弹出的导出对话框中，配置以下选项：
   - **文件名**：设置导出文件的名称。
   - **导出类型**：选择“CSV”格式。
   - **导出范围**：选择导出当前页数据或全部数据。

{{% load-img "/img/management/data-explorer/export-csv/image-2.png" %}}

4. **完成导出**：点击「导出」按钮，系统将生成 CSV 文件并自动下载到本地。

{{% load-img "/img/management/data-explorer/export-csv/image-3.png" %}}

## 注意事项

- 导出的数据受当前时间范围、搜索条件和过滤条件的限制，如需导出完整数据，请确认已移除不必要的筛选条件。
- 选择「全部数据」导出时，数据量较大可能导致导出耗时较长，建议先通过过滤条件缩小数据范围。
- 导出的 CSV 文件编码为 UTF-8，使用 Excel 打开时如出现中文乱码，可尝试以 UTF-8 编码方式导入。
