# 字符串字段类型

字符串字段类型包含文本值或者从文本派生的值。  
如何选择 `text` 与 `keyword`、设计多字段，请优先阅读映射章节的 Guide。

## 字段类型总览

字符串字段类型包含文本值或者从文本派生的值。下表列出了 Easysearch 支持的所有字符串字段类型。

| 字段数据类型 | 描述                                     |
| ------------ | ---------------------------------------- |
| keyword      | 未经分析的字符串。适用于精确值搜索。     |
| text         | 经过分词器分析的字符串。适用于全文搜索。 |
