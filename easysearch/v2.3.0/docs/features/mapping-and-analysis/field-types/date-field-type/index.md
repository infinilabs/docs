# 日期字段类型

日期字段类型包含可以使用不同日期格式进行格式化的日期值。  
如何选择日期格式、处理时区，请优先阅读映射章节的 Guide。

## 字段类型总览

下表列出了 Easysearch 支持的所有日期字段类型。
| 字段数据类型 | 描述 |  
|--------------------|--------------------------------------|  
| **date** | 以毫秒精度存储的日期值。 |  
| **date_nanos** | 以纳秒精度存储的日期值。 |
