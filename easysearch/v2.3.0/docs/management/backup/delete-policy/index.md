# 删除策略

当备份策略不再需要时，可以将其删除。

## 操作步骤

1. **进入备份策略页面**：登录 Easysearch UI，左侧导航栏点击「备份管理」，切换至「策略」标签页。
2. **触发删除**：找到目标策略（如 `test_strategy`），点击「操作」列的更多按钮，选择「删除」。

{{% load-img "/img/management/backup/delete-policy/image-1.png" %}}

3. **确认执行**：在弹出提示框中核对名称，点击「确定」。

{{% load-img "/img/management/backup/delete-policy/image-2.png" %}}

4. **验证结果**：操作完成后，目标策略从列表移除，仅剩余未删除的策略。

{{% load-img "/img/management/backup/delete-policy/image-3.png" %}}

## 注意事项

- 删除策略仅停止自动备份计划，**不会删除**已生成的快照数据。
- 操作前请确认业务不再依赖该策略的自动备份，避免数据无法按预期自动备份。
- 策略删除后可随时重新创建，已存在的快照需手动管理。
