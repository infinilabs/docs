# 滚动数据流

通过滚动数据流，可以为数据流创建新的后备索引，并将后续写入切换到新的写入索引。

## 操作步骤

1. **进入数据流页面**：在左侧导航栏点击「数据流」，进入数据流列表页面。

2. **选择滚动操作**：找到需要滚动的目标数据流，点击「操作」列的更多按钮（`...`），在弹出菜单中选择「滚动」。

{{% load-img "/img/management/data-stream/rollover-data-stream/image-1.png" %}}

3. **确认滚动**：在弹出的确认提示框中，点击「确定」按钮，系统将执行数据流滚动操作。

{{% load-img "/img/management/data-stream/rollover-data-stream/image-2.png" %}}

4. **查看滚动结果**：滚动成功后，数据流会生成新的后备索引，后续写入将进入新的写入索引。

## 注意事项

- 滚动数据流会改变后续写入的目标后备索引，不会删除已有后备索引中的数据。
- 执行滚动前请确认数据流状态正常，且关联模板配置符合当前写入需求。
- 如业务依赖固定后备索引名称，请在滚动前确认相关引用不会受到影响。
