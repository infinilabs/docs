# 新增数据流

通过新增数据流，可以按时间戳字段持续写入和管理时序数据。

## 操作步骤

1. **进入数据流页面**：在左侧导航栏点击「数据流」，进入数据流列表页面。

2. **打开新建数据流弹窗**：点击页面右上角的「+ 新增」按钮，弹出「新建数据流」配置窗口。

{{% load-img "/img/management/data-stream/create-data-stream/image-1.png" %}}

3. **配置数据流信息**：
   - 填写必填项「名称」，自定义数据流标识（如示例中的 `test_data_stream`）。
   - （可选）填写「模板名称」，用于指定或生成数据流关联的索引模板。
   - 填写「时间戳字段」，指定数据流用于时间排序的字段（默认值为 `@timestamp`）。
   - （可选）填写「优先级」，用于控制生成模板与其它模板同时匹配时的应用顺序。
   - （可选）填写「主分片数」「副本分片数」，按需配置后备索引的分片数量。

4. **保存数据流**：配置完成后，点击「保存」按钮，系统将自动创建数据流。

{{% load-img "/img/management/data-stream/create-data-stream/image-2.png" %}}

5. **查看创建结果**：数据流创建成功后，将在列表中展示该数据流的完整信息，包括名称、状态、关联模板和索引个数等。示例中新增的 `test_data_stream` 状态为 **Green**。

{{% load-img "/img/management/data-stream/create-data-stream/image-3.png" %}}

## 注意事项

- 数据流名称不可与已有数据流重复，否则创建将失败。
- 时间戳字段用于数据流写入和滚动管理，建议使用稳定的时间字段（如 `@timestamp`）。
- 主分片数创建后不可修改，副本分片数可按需调整，请根据数据量和查询负载提前规划。
- 数据流创建后会生成或关联对应的索引模板，并创建后备索引用于承载实际数据。
