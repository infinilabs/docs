# 复制管道

通过复制管道，可以基于已有摄取管道快速创建一份新的管道配置。

## 操作步骤

1. **进入管道管理页面**：在左侧导航栏点击「管道管理」，进入摄取管道列表页面。

2. **选择复制操作**：找到需要复制的目标管道，点击「操作」列的更多按钮（`...`），在弹出菜单中选择「复制」。

{{% load-img "/img/management/pipeline-management/copy-pipeline/image-1.png" %}}

3. **调整复制后的配置**：系统将进入「新建摄取管道」页面，并自动带出原管道的处理器配置。填写新的管道名称（如示例中的 `test_pipeline_copy`），按需修改描述和处理器配置。

4. **保存管道**：确认配置无误后，点击「保存」按钮。

{{% load-img "/img/management/pipeline-management/copy-pipeline/image-2.png" %}}

5. **查看复制结果**：管道保存成功后，将返回管道管理列表。示例中新增的 `test_pipeline_copy` 已展示在列表中，处理器数量为 `1`。

{{% load-img "/img/management/pipeline-management/copy-pipeline/image-3.png" %}}

## 注意事项

- 复制管道时需要为新管道填写唯一名称，不能与已有管道重复。
- 复制操作会带出原管道的处理器配置，保存前请检查是否需要调整处理器顺序或字段配置。
- 新管道保存后不会自动替换业务中已使用的原管道，如需切换请同步更新业务侧引用。
