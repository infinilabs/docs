# 管道模拟测试

通过管道模拟测试，可以在不写入真实索引的情况下验证摄取管道的处理结果。

## 操作步骤

1. **进入管道管理页面**：在左侧导航栏点击「管道管理」，进入摄取管道列表页面。

2. **进入管道详情页**：点击需要测试的目标管道名称，进入管道详情页面。

{{% load-img "/img/management/pipeline-management/simulate-pipeline/image-1.png" %}}

3. **切换到模拟测试**：在管道详情页中，点击「模拟测试」标签页。

4. **输入测试文档**：在「输入测试文档」编辑器中填写用于测试的文档 JSON。示例中输入了 `_index`、`_id` 和 `_source.test_append` 字段。

5. **执行测试**：点击「测试」按钮，系统将使用当前管道配置对测试文档执行模拟处理。

{{% load-img "/img/management/pipeline-management/simulate-pipeline/image-2.png" %}}

6. **查看测试结果**：测试完成后，可在右侧「匹配结果」区域查看处理后的文档内容。示例中 `test_append` 字段追加了 `test_append` 值，并生成了 `_ingest.timestamp`。

{{% load-img "/img/management/pipeline-management/simulate-pipeline/image-3.png" %}}

## 注意事项

- 模拟测试不会将文档写入真实索引，仅用于验证管道处理逻辑。
- 测试文档需使用合法 JSON，且字段结构应尽量贴近真实写入数据。
- 如果测试结果不符合预期，请检查处理器顺序、字段名称和条件配置。
