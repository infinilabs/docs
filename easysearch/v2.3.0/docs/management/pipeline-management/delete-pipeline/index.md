# 删除管道

当摄取管道不再使用时，可以通过管道管理页面进行删除。

## 操作步骤

1. **进入管道管理页面**：在左侧导航栏点击「管道管理」，进入摄取管道列表页面。

2. **选择删除操作**：找到需要删除的目标管道，点击「操作」列的更多按钮（`...`），在弹出菜单中选择「删除」。

{{% load-img "/img/management/pipeline-management/delete-pipeline/image-1.png" %}}

3. **确认删除**：在弹出的确认提示框中，点击「确定」按钮，系统将执行删除操作。

{{% load-img "/img/management/pipeline-management/delete-pipeline/image-2.png" %}}

4. **查看删除结果**：删除成功后，该管道将从管道管理列表中移除。示例中 `test_pipeline_copy` 已不再显示在列表中。

{{% load-img "/img/management/pipeline-management/delete-pipeline/image-3.png" %}}

## 注意事项

- 删除管道后无法在 UI 中直接恢复，如需再次使用请重新创建或导入配置。
- 如果该管道已被索引写入流程引用，删除后相关写入请求可能失败，请提前确认业务侧引用情况。
- 删除操作仅删除管道配置，不会删除已写入索引中的文档数据。
