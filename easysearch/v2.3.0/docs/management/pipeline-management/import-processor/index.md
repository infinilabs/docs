# 导入处理器

通过导入处理器，可以将已有的管道处理器 JSON 配置快速应用到新建摄取管道中。

## 操作步骤

1. **进入新建摄取管道页面**：在「管道管理」页面点击「+ 新增」，进入「新建摄取管道」页面。

2. **填写管道基础信息**：填写必填项「名称」（如示例中的 `test_pipeline_2`），按需填写「描述」。

3. **打开导入处理器窗口**：在「处理器」区域右侧，点击「导入处理器」按钮。

{{% load-img "/img/management/pipeline-management/import-processor/image-1.png" %}}

4. **粘贴处理器 JSON 配置**：在弹出的「导入处理器」窗口中，将管道处理器的 JSON 配置粘贴到编辑器中。

5. **导入配置**：确认 JSON 内容无误后，点击「导入并覆盖」按钮，系统将使用导入内容覆盖当前处理器列表。

{{% load-img "/img/management/pipeline-management/import-processor/image-2.png" %}}

6. **确认导入结果**：导入成功后，处理器列表中将展示对应处理器。示例中导入后展示 `Append` 处理器。

7. **保存管道**：确认处理器配置无误后，点击「保存」按钮。

{{% load-img "/img/management/pipeline-management/import-processor/image-3.png" %}}

8. **查看创建结果**：管道创建成功后，将返回管道管理列表。示例中新增的 `test_pipeline_2` 已展示在列表中，处理器数量为 `1`。

{{% load-img "/img/management/pipeline-management/import-processor/image-4.png" %}}

## 注意事项

- 导入内容需为合法 JSON，且结构需符合摄取管道处理器配置格式。
- 点击「导入并覆盖」会覆盖当前页面已有的处理器列表，请在导入前确认是否需要保留现有配置。
- 导入后建议检查处理器顺序和字段配置，确认无误后再保存管道。
