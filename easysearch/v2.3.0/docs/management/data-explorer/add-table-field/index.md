# 新增表格字段

可以将字段添加到数据探索的表格视图中进行展示。

## 操作步骤

1. **进入数据探索页面**：在左侧导航栏点击「数据探索」，进入数据探索页面。
2. **添加字段到表格**：在左侧字段列表，鼠标悬停在字段上，点击「+」图标即可。
